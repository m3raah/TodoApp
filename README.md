Краткое описание проекта
Тема проекта: Менеджер задач (Todo List).
Какие дополнительные задания реализованы: Проект успешно разделен на два независимых слоя согласно архитектурному паттерну:
YourProject.Core (Библиотека классов) — содержит модели данных (TodoItem), интерфейсы репозиториев, слой хранения данных в оперативной памяти (InMemoryTodoRepository) и сервисы бизнес-логики (TodoService).
YourProject.WinForms (Приложение Windows Forms) — отвечает исключительно за графический интерфейс пользователя. Связь между окнами реализована через передачу объектов и использование свойств в конструкторах форм. Отображение коллекции объектов выполнено динамически через элемент DataGridView.
Какие трудности возникли: Основная сложность заключалась в первоначальном распределении пространств имен (namespace) и устранении циклической зависимости при подключении ссылки на проект YourProject.Core внутри интерфейсного проекта YourProject.WinForms.
